# Typerion TD 🌱

タイピングでワードを打ち込み、植物ユニットを召喚して敵から拠点を守る **タイピング × タワーディフェンス** です。  
ビルド不要の静的サイトで、成績はブラウザの **LocalStorage** にだけ保存されます（アカウント不要）。

---

## 遊び方（かんたん）

1. **ステージ選択**でステージを選ぶ  
2. 画面中央の**ワードをタイプ**する  
3. 打ち終えるとユニットが出現し、**自動で戦闘**  
4. **敵拠点**を破壊するとクリア、**自拠点**の HP が 0 になるとゲームオーバー  
5. クリアで **成熟度** が増え、拠点の見た目・強さが変化（🌱 → 🌿 → 🌳）

戦闘中は **`Tab + 1` / `2` / `3`**（または画面のボタン）で **Tier（初級・中級・上級）** を切り替えます。Tier が高いほど強いユニットになりやすい一方、打つ文字数も増えます。

---

## 主な内容

| 内容 | 説明 |
|------|------|
| タイピング | ミスが多いと、その召喚でユニットの初期 HP が下がる |
| Tier | Tab+数字で切替。単語の長さ・ユニットの傾向が変わる |
| 成熟度・拠点 | プレイで蓄積。拠点の成長に反映 |
| 記録 | ヒートマップ、苦手キー、履歴、ベストスコアなど（**記録**画面） |

---

## Tier のイメージ

| Tier | 操作 | 単語の目安 | ユニットの傾向 |
|------|------|------------|----------------|
| 1 🌱 | Tab+1 | 短め（ホームポジ中心） | 速い・薄め |
| 2 🌿 | Tab+2 | 中くらい（英単語など） | バランス型 |
| 3 🌲 | Tab+3 | 長め・コード片など | 遅い・厚め |
